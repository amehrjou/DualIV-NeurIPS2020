# DeepGMM

Code for DualIV experiments for 
**Deep Generalized Method of Moments for Instrumental Variable Analysis**  
Andrew Bennett, Nathan Kallus, Tobias Schnabel  
<https://arxiv.org/abs/1905.12495>

### Instructions for DualIV demand experiments
1. Unzip demand.zip in DeepGMM/data
2. Run python demand_experiment.py (Exisitng results present in DeepGMM/resuts will be overwritten)
3. Results can be found in init_var_0.1_epochs_3000_deepgmm_results.json
